  <h2>ソーダン！</h2>
  <h3>相談は気軽に。回答は真面目に。</h3>
  <p>ソーダン！は悩み相談サービスです。</p>
  <h3>主な機能</h3>
  <ul>
    <li>Twitter認証で利用</li>
    <li>相談は匿名でできる</li>
    <li>回答はTwitterIdを表示する</li>
    <li>回答時には他の回答は見れない</li>
    <li>回答はn人まで受け付け、n人回答した時点で相談内容と回答がpublicになる</li>
    <li>相談者は自分の相談した回答について、n人分の回答が集まらなくても閲覧できる</li>
    <li>とりあえずテスト中なのでn=3にしておく</li>
  </ul>
